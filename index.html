Data-Blocks Documentation

This is a Scratch extension that I am making which will allow for more data manipulation.

explode [] for [] return ()
(reporter)

This block will take the first parameter after explode, split it by the second parameter (after for) and return the nth part of the split string. For example explode [I like pie] for [ ] return (1) will return "like" because it starts counting at zero.
